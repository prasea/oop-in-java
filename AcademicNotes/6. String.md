## **1. Introduction to Strings in Java**

### **1.1 What is a String?**

A **String** in Java is a **sequence of characters** enclosed within double quotes.

**Exam Definition:**

> A String is an object that represents a sequence of characters.

---

### **Key Characteristics of Strings**

* Strings are **objects**, not primitive data types
* Strings are **immutable** (cannot be changed once created)
* Java provides extensive **String manipulation methods**

---

### **Example**

```java
String name = "Core Java";
```

---

## **2. Creation of Strings**

Strings can be created in **two ways**:

---

### **2.1 Using String Literal**

```java
String s1 = "Java";
```

Stored in **String Constant Pool (SCP)**
If another string with same value is created, it refers to the same memory.

---

### **2.2 Using `new` Keyword**

```java
String s2 = new String("Java");
```

Creates a **new object in heap memory**

---

### **Difference Between Literal and `new` Keyword**

| String Literal           | Using new          |
| ------------------------ | ------------------ |
| Memory efficient         | Creates new object |
| Stored in SCP            | Stored in Heap     |
| Same value shares memory | No sharing         |


---

## **3. String Concatenation**

### **Meaning**

**Concatenation** means **joining two or more strings**.

---

### **3.1 Using `+` Operator**

```java
String s = "Java" + " Programming";
```

---

### **3.2 Using `concat()` Method**

```java
String s = "Java".concat(" Programming");
```

---

### **Example**

```java
String first = "Core";
String second = "Java";
System.out.println(first + " " + second);
```

---

**Exam Note:**
Each concatenation creates a **new String object** due to immutability.

## **6. Character Extraction from String**

Used to retrieve characters from a string.

### **6.1 `charAt()` Method**

```java
String s = "Java";
char c = s.charAt(1);   // 'a'
```

---

### **6.2 `getChars()` Method**
The getChars() method in Java copies a specific range of characters from a source string (or StringBuffer/StringBuilder) into a destination character array. 

Syntax: `public void getChars(int srcBegin, int srcEnd, char[] dst, int dstBegin)`

```java
String sourceString = "Welcome to Java";
// Create a destination character array large enough to hold the characters
char[] destinationArray = new char[10]; 

// Copy characters from index 4 to 10 (inclusive of 4, exclusive of 11) into destinationArray starting at index 0
sourceString.getChars(4, 11, destinationArray, 0); 

System.out.println(destinationArray); // Output: me to Ja
```

## **7. String Comparison**

String comparison checks **content or reference**.

- Using `equals()` Method Compares **content**
- Using `==` Operator Compares **reference (memory location)**

```java 
String s1 = new String("Java");
String s2 = new String("Java");
String s3 = "Java";
String s4 = "Java";

System.out.println(s1 == s2);      // Output: false (s1 and s2 are different objects in memory)
System.out.println(s1.equals(s2)); // Output: true (s1 and s2 have the same content)

System.out.println(s3 == s4);      // Output: true (s3 and s4 refer to the same object in the String pool due to optimization)
System.out.println(s3.equals(s4)); // Output: true (s3 and s4 have the same content)
```

---

### **7.3 `compareTo()` Method**

```java
String s1 = "Apple";
String s2 = "Ball"; 
System.out.println(s1.compareTo(s2));
```

* Returns `0` → equal
* Positive → s1 > s2
* Negative → s1 < s2

## **10. StringBuffer Class**

### **10.1 Meaning**

**StringBuffer** is a class used to create **mutable strings**.

**Exam Definition:**

> StringBuffer represents a growable and modifiable sequence of characters.

---

### **Key Features**

* Mutable
* Thread-safe (synchronized)
* Faster for frequent modifications

---

### **Example**

```java
StringBuffer sb = new StringBuffer("Java");
sb.append(" Programming"); //sb = "Java Programming"
sb.insert(0, "Core "); // sb = "Core Java Programming"
sb.delete(17, 21); // sb = "Core Java Program"
System.out.println(sb);
```

---

### **Common StringBuffer Methods**

| Method    | Purpose            |
| --------- | ------------------ |
| append()  | Adds text          |
| insert()  | Inserts text       |
| delete()  | Removes characters |
| reverse() | Reverses string    |

---

### **String vs StringBuffer**

| String             | StringBuffer |
| ------------------ | ------------ |
| Immutable          | Mutable      |
| Thread-safe        | Thread-safe  |
| Slower for changes | Faster       |
